
## Projeto Delivery IA – Recomendações Personalizadas

Este projeto simula um sistema de recomendação inspirado no iFood, utilizando inteligência artificial e machine learning para sugerir restaurantes com base no perfil de cada cliente. Ele combina filtragem colaborativa com preferências de conteúdo para entregar sugestões relevantes e personalizadas.

---

## Tecnologias utilizadas:
- Python 3.12
- Pandas e NumPy – manipulação de dados
- Scikit-learn – cálculo de similaridade
- Streamlit – dashboard interativo

---

## Como rodar localmente:
1. Clone o repositório:
   git clone https://github.com/tineslee/projeto-delivery-ia.git
   cd projeto-delivery-ia

2. Crie e ative o ambiente virtual:
   python3 -m venv venv
   source venv/bin/activate

3. Instale as dependências:
   pip install -r requirements.txt

4. Rode o app:
   streamlit run projeto_delivery_ia.py

---

## Funcionalidades:
- Geração de dados sintéticos com perfis de clientes e restaurantes
- Sistema de recomendação híbrido (colaborativo + conteúdo)
- Nomes fictícios de restaurantes gerados automaticamente
- Dashboard interativo com seleção de cliente e visualização de categorias sugeridas
